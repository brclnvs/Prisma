# 🌟 Welcome to Prisma! 🌟

**Prisma** is an operating system aiming to provide a **stable and reliable desktop experience** for everyone.

## ✨ Prisma's Goal

Prisma strives to be an OS that simply works, day in and day out. The system focuses on being:

* **Rock-solid:** Designed for dependability, allowing users to count on it for daily tasks.
* **User-friendly:** Simple and intuitive to use from the very beginning.
* **A joy to use:** Providing a smooth and pleasant experience every time it boots up.

---

## 🚧 A Glimpse Behind the Scenes: Pre-Alpha!

Currently, Prisma is in its very early **pre-alpha** stages. This means its foundations are being laid, with significant effort focused on:

* **Squashing bugs:** Identifying and fixing glitches to ensure smooth operation.
* **Essential features:** Implementing core functionalities necessary for a reliable desktop.

The system is built using `bootc`, which helps in prioritizing stability from the ground up.
