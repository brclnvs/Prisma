# 🌟 Bem-vindo ao Prisma! 🌟

**Prisma** é um sistema operacional que busca oferecer uma **experiência de desktop estável e confiável** para todos.

## ✨ O Objetivo do Prisma

O Prisma se esforça para ser um sistema operacional que simplesmente funciona, dia após dia. O sistema se concentra em ser:

* **Sólido como uma rocha:** Projetado para ser confiável, permitindo que os usuários contem com ele para suas tarefas diárias.
* **Fácil de usar:** Simples e intuitivo desde o início.
* **Um prazer de usar:** Proporcionando uma experiência suave e agradável a cada inicialização.

---

## 🚧 Uma Visão dos Bastidores: Pré-Alpha!

Atualmente, o Prisma está em seus estágios iniciais de **pré-alpha**. Isso significa que suas fundações estão sendo lançadas, com um esforço significativo concentrado em:

* **Correção de bugs:** Identificando e corrigindo falhas para garantir uma operação tranquila.
* **Recursos essenciais:** Implementando as funcionalidades principais necessárias para um desktop confiável.

O sistema é construído usando `bootc`, o que ajuda a priorizar a estabilidade desde o início.
